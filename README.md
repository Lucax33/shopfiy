# VELLA — Shopify Custom Theme

A clean girl / soft luxury dropshipping store theme built for elegance, comfort, and high performance.

## Design Ethos
- **Aesthetic:** Quiet luxury, warm minimalism, editorial fashion.
- **Typography:** Cormorant Garamond (Headings) + DM Sans (Body).
- **Colors:** Warm neutrals with gold accents (#C4A265).

## Tech Stack
- **Shopify Liquid:** Optimized for Online Store 2.0.
- **Vanilla CSS:** Custom properties and fluid typography with `clamp()`.
- **Vanilla JS:** Minimal footprint, no jQuery, high-performance animations with IntersectionObserver.

## Theme Features
- Dynamic Announcement Bar.
- Sticky Header with Mobile Navigation Drawer.
- Full-width Hero Banner with Staggered Animations.
- Integrated AJAX Cart Drawer with Free Shipping Progress Bar.
- Customizable Homepage Sections:
  - Featured Products (3-column/4-column responsive).
  - Brand Statement.
  - Category Grid (2x2).
  - Instagram/UGC Grid.
  - Newsletter Signup.
- Advanced Product Pages:
  - Multi-image Gallery with Thumbnails.
  - Interactive Color and Size Selectors.
  - AJAX Add to Cart.
  - Trust Badges and Tabbed Content.
- Informative Content Pages:
  - FAQ with Accordion.
  - Balanced About Us layout.
  - Styled Contact Form.

## Performance
- Target Lighthouse Score: 90+
- Performance-first approach to assets and loading.
- Native Shopify image_tag for lazy loading and responsive sizes.
